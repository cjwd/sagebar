## What is this?
A simple WP plugin that tells you what main and base template is being used by a [Sage 8 WP Theme](https://github.com/roots/sage/tree/8.5.3)

## Disclaimer
__Only works for Sage 8 - 8.5.3__

## Usage
1. Install the plugin
2. Enable and configure the bar at Tools > Sagebar Options

The plugin also includes a simple utility function for var_dump which is available if
WP_DEBUG is set to true.

`dump($var)`

or

`dump($var)`

to dump the variable and stop execution of the script.
